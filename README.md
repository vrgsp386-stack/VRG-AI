# VRG-AI — Smart Command Center

> An AI-powered personal website with a search engine, resume builder, QR generator, tools hub and idea generator.

🌐 **Live site:** [vrg-ai.vercel.app](https://vrg-ai.vercel.app)

---

## Features

| Section | Description |
|---|---|
| 🔍 AI Search Engine | Ask anything — get instant AI-powered answers |
| 🛠️ Tools Hub | Canva, Figma, Web Tech, ChatGPT, Gamma, Notion, n8n, Perplexity |
| 💬 Ask AI Anything | Questions, Problem Solving, Idea Generation, Content Creation, Business Advice |
| 🔲 QR Generator | Paste any link → instant QR code → download as PNG |
| 📄 Resume Builder | Fill details, choose theme colour & font → download as PDF |
| 💡 Ideas & Solutions | Startup ideas, marketing strategies, business plans, problem solving |

---

## Tech Stack

- **HTML / CSS / JavaScript** — Single file, zero dependencies
- **Claude AI API** — Powers the AI search and answer engine
- **QRCode.js** — QR code generation
- **html2canvas** — Resume PDF download
- **Vercel** — Free hosting and deployment
